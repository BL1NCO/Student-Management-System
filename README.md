# 🎓 Student Management System

An OOP-based CRUD application for managing student records such as names, IDs, grades, and courses.

## 🚀 Features

- Add student records
- View all students
- Update student information
- Delete student records
- Search students by ID
- Persistent data storage

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- JSON
- File Handling

## 📂 Project Structure

```text
student-management-system/
│
├── main.py
├── student.py
├── students.json
└── README.md
```

## ▶️ How to Run

```bash
python main.py
```
